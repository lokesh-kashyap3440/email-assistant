# Email Assistant

AI-powered email assistant for your terminal and web — summarizes, categorizes, extracts action items, and suggests replies. Uses the **OpenCode server API** as the LLM backend.

## Web UI

Open the app in your browser to access the dashboard.

## CLI Usage

```bash
pip install email-assistant
email-assistant setup --creds path/to/credentials.json
email-assistant inbox
```
